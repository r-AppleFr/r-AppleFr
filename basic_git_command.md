# Basic Git Command

## Initiate a repo
``git init``
## Show your changes
``git status``
## Add your local changes
``git add .``
## Commit your changes with a message
``git commit -m "Your message"``
## Push your code to GitHuv
``git push``
## Change branch
``git checkout NAMEOFBRANCH``
